
####Usage

#####Install gems
bundle

#####Update Github credentials in downloader.rb
github_email = 'YOUR GITHUB EMAIL ADDRESS'

github_password = 'YOUR GITHUB PASSWORD'

#####Execute script
./downloader.rb
